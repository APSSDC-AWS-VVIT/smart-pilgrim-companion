# Slide 2 Notes: Problem Statement + Motivation

## Speaker Notes

Pilgrimage planning is often fragmented. Users need temple details, routes, budgets, nearby places, schedules, and visit guidance, but these are usually spread across different sources. Our project solves this by creating a single platform for temple discovery and travel planning.

## Viva Talking Points

- Problem is fragmented planning.
- Users need information before travel.
- Platform combines temple, route, budget, and planner support.
